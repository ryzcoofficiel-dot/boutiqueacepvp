# Boutique ACEPVP - version refaite

## Ce que fait cette version
- panier PayPal propre pour les packs AceCoins
- plusieurs packs payés en une seule transaction
- VIP affiché comme achat en AceCoins (2000 AC)
- véhicules affichés comme catalogue récupéré en jeu
- ticket Discord pour récupération des AceCoins après paiement
- interface responsive refaite depuis une base propre

## Fonctionnement
1. Le joueur ajoute un ou plusieurs packs AceCoins au panier.
2. Il paie le total via PayPal.
3. Il ouvre un ticket Discord avec sa preuve PayPal.
4. Le staff crédite les AceCoins.
5. Le joueur utilise ses AceCoins en jeu pour le VIP et les véhicules.

## Fichiers à vérifier
- `index.html` : client-id PayPal
- `script.js` : données produits / véhicules / Discord
- `images/` : logos et visuels

Cette version ne montre plus de popup de succès au chargement.
